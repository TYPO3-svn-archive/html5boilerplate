#
# What it this project about?
#

HTML5Boilerplate is a collection of best practices of html, css and javascript collections
for a cross browser, fast, robust and future-proof site.

This extension makes HTML5Boilerplate features available for TYPO3.

If you want to know more about the HTML5Boilerplate project please visit
the  project website on http://html5boilerplate.com/

If you want to get a great overview about the HTML5Boilerplate watch the videos
"Paul Irish on HTML5 Boilerplate" (http://www.youtube.com/watch?v=qyM37XKkmKQ),
"HTML5 Boilerplate Walkthrough" (http://www.youtube.com/watch?v=oDlsOyPKUTM) or
have a look at http://html5boilerplate.com/#videos

If you want to know more about the TYPO3 integration of HTML5Boilerplate please 
have a look at the Forge project "HTML5 Boilerplate" (http://forge.typo3.org/projects/extension-html5boilerplate)

Thanks to all contributers of HTML5Boilerplate and the TYPO3 extension html5boilerplate!
